# Flex box layout

## For 2 columns layout

You can use `justify-content: between;`.

<div style={{display: `flex`, flexWrap: `wrap`, justifyContent: `space-between`}}>
    <div style={{height: 140, width: `calc(50% - 15px)`, background: `#999`}}></div>
    <div style={{height: 140, width: `calc(50% - 15px)`, background: `#999`}}></div>
</div>

## For 3 or more columns layout

But, if you use `justify-content: between;` for 3 or more columns layout. Sometimes layout will be weired like the following:

<div style={{display: `flex`, flexWrap: `wrap`, justifyContent: `space-between`}}>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, background: `#999`}}></div>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, background: `#999`}}></div>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, background: `#999`}}></div>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, background: `#999`, marginTop: 30}}></div>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, borderStyle: `dashed`, borderWidth: 3, borderColor: `#999`, marginTop: 30, display: `flex`, alignItems: `center`, justifyContent: `center`}}>BLANK</div>
    <div style={{height: 140, width: `calc(33.33% - 15px)`, background: `#999`, marginTop: 30}}></div>
</div>

## Solution

You can resolve flex box layout with negative margin for parent box.

<div style={{display: `flex`, flexWrap: `wrap`, justifyContent: `flex-start`, margin: `0 -15px 30px`}}>
    <div style={{height: 140, width: `calc(100% / 3 - 30px)`, background: `#999`, margin: `0 15px`}}></div>
    <div style={{height: 140, width: `calc(100% / 3 - 30px)`, background: `#999`, margin: `0 15px`}}></div>
    <div style={{height: 140, width: `calc(100% / 3 - 30px)`, background: `#999`, margin: `0 15px`}}></div>
    <div style={{height: 140, width: `calc(100% / 3 - 30px)`, background: `#999`, margin: `30px 15px 0`}}></div>
    <div style={{height: 140, width: `calc(100% / 3 - 30px)`, background: `#999`, margin: `30px 15px 0`}}></div>
</div>


<Tabs>
<TabItem value="scss" label="SCSS">

```scss
.c-flex {
    // Add negative margin half of gap size to horizontaly
    margin: 0 -15px;

    &__col {
        height: 140px;

        // Add margin half of gap size to horizontaly
        margin: 0 15px;

        // 100% divided by the number of column size and subtracted by gap size
        width: calc(100% / 3 - 30px);

        &:nth-child(n+4) {
            margin-top: 30px;
        }
    }
}
```

</TabItem>

```html
    <div class="c-flex__col"></div>
</div>
```

# Options

> Make option tags

```php
{{ options source=$array selected=$selected }}
```

Make `<option>` from `$array`.

## Example

```php title="Template"
/*
$values = array(
    'foo' => 'bar',
    'baz' => 'qux'
);
$selected = 'foo';
*/

<select name="select">
    {{ options source=$values selected=$selected }}
</select>
```

```html title="Output"
<select name="select">
    <option value="foo" selected="selected">bar</option>
    <option value="baz">qux</option>
</select>
```

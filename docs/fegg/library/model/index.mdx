## How to use

`Model` is already imported on `Controller`, so you can call without include.

```php title="Controller"
$model = new Model(false, array(
  'table'   => 'cms_news',
  'id'      => 'news_id',
  'visible' => 'visible',
));

$records = $model->find();
```

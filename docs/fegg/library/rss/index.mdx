# Introduction

Providing some functions for analyzing RSS document

## How to use

You can load `RSS` class through `Application::getClass` method.

```php title="Controller"
$rss = $this->getClass('Tool/RSS', 'https://example.com/feed');
$posts = $rss->fetchAll(10);
```

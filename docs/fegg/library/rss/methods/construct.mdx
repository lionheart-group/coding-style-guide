# RSS::\_\_construct

Start loading RSS feed

## Description

```php
public self RSS::__construct( string $url )
```

## Arguments

| Name | Data type | Default | Remarks      |
| ---- | --------- | ------- | ------------ |
| $url | string    |         | RSS Feed URL |

## Example

```php title="Controller"
$rss = $this->getClass('Tool/RSS', 'https://example.com/feed');
```

Display converted image


## Description

```php
public void Trim::view()
```


## Example

```php title="Controller"
$trim = $this->getClass('Tool/RSS', 'https://example.com/feed');
$trim->setImage('path/to/image.jpg', 'path/to/converted.jpg');

$trim->resizeTrim(300, 300, 'm');
$trim->view();
```

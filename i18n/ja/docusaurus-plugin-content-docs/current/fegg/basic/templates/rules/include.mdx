# インクルード

> include、include head、head、include foot、foot

## 説明

```php
{{ include 'foo/bar' }}
```

```php
{{ include head 'foo/bar' }}
{{ head }}
```

```php
{{ include foot 'foo/bar' }}
{{ foot }}
```

別のテンプレートをインクルードします。

`{{ include }}`はテンプレートを読み込み、記述された場所に表示しますが、`{{ include head }}`と`{{ include foot }}`はテンプレートを読み込み、`{{ head }}`と`{{ foot }}`の位置に表示します。

## 例

```php title="テンプレート - child.tpl"
<p>このファイルはchild.tplです</p>
```

```php title="テンプレート - parent.tpl"
<p>このファイルはparent.tplです</p>
{{ include 'parts' }}
```

```html title="出力"
<p>このファイルはchild.tplです</p>
<p>このファイルはparent.tplです</p>
```

`{{ include head }}`と`{{ include foot }}`を使用する場合。

```php title="テンプレート - header.tpl"
<link rel="stylesheet" type="text/css" href="path/to/style.css">
```

```php title="テンプレート - footer.tpl"
<script type="text/javascript" src="path/to/script.js"></script>
```

```php title="テンプレート - parent.tpl"
{{ include head 'header' }}
{{ include foot 'footer' }}

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>ページタイトル</title>

    {{ head }}

</head>
<body>
    何かの内容

    {{ foot }}

</body>
</html>
```

```html title="出力"
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>ページタイトル</title>

    <link rel="stylesheet" type="text/css" href="path/to/style.css">

</head>
<body>
    何かの内容

    <script type="text/javascript" src="path/to/script.js"></script>

</body>
</html>
```

# Trim::resize

比率を保持して画像をリサイズ

## 説明

```php
public void Trim::resize( number $w, number $h )
```

## 引数

| 名前 | データ型 | デフォルト | 備考               |
| ---- | -------- | ---------- | ------------------ |
| $w   | number   |            | 変換後の画像の幅   |
| $h   | number   |            | 変換後の画像の高さ |

## 例

```php title="Controller"
$trim = $this->getClass('Tool/Trim');
$trim->setImage('path/to/original.jpg', 'path/to/resized.jpg');
$trim->resize(800, 600);
$trim->create();
```

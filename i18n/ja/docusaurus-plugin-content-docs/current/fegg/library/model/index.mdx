# 概要

import Badge from "../../../../../../../src/components/Badge";

> LHプロジェクト用拡張データベースライブラリ

## 使用方法

`Model`は`Controller`に既にインポートされているため、includeなしで呼び出すことができます。

```php title="コントローラー"
$model = new Model(false, array(
  'table'   => 'cms_news',
  'id'      => 'news_id',
  'visible' => 'visible',
));

$records = $model->find();
```

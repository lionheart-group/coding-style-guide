# Model::eraseWhere

> 指定された条件でレコードの物理削除を実行

## 説明

```php
public void Model::eraseWhere( string $where, array $wh_data )
```

## 引数

| 名前     | データ型 | デフォルト | 備考           |
| -------- | -------- | ---------- | -------------- |
| $where   | string   |            | 削除条件       |
| $wh_data | array    |            | 削除条件データ |

## 例

```php title="Controller"
$model = new Model(false, array('table' => 'cms_news'));
$model->eraseWhere('status = ?', [0]);
```

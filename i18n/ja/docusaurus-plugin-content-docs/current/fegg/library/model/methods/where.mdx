# Model::where

> Whereパラメータを設定

## 説明

```php
public self Model::where( string $query, [ array $data = [], string $andOr = 'AND' ] )
```

## 引数

| 名前   | データ型 | デフォルト | 備考               |
| ------ | -------- | ---------- | ------------------ |
| $query | string   |            | WHERE条件クエリ    |
| $data  | array    | []         | バインドデータ     |
| $andOr | string   | 'AND'      | 論理演算子         |

## 例

```php title="Controller"
$model = new Model(false, array(
    'table'   => 'cms_news',
    'id'      => 'news_id',
    'visible' => 'visible',
));

$model->where('status = ?', [1]);
```

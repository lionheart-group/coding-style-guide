# はじめに

ファイルのアップロード/検証のための各種関数を提供します

## 使用方法

`Application::getClass`メソッドを通じて`Upload`クラスを読み込むことができます。

```php title="Controller"
$upload = $this->getClass('Tool/Upload');

$upload->upload('file');
```

# はじめに

ファイル/ディレクトリ管理のための各種関数を提供します

## 使用方法

`Application::getClass`メソッドを通じて`File`クラスを読み込むことができます。

```php title="Controller"
$file = $this->getClass('Tool/File');

$file->copyAll('path/to/foo', 'path/to/bar');
```

# 動画ファイルの圧縮

## 推奨サイズ

- **背景画像として表示する音声なし動画（読み込み速度を重視）**
  - 推奨ファイルサイズ：**1MB以下（理想）から最大3MB**
  - 注意：読み込み速度が重要です。これらの動画はページの初期読み込みに影響するため、可能な限り軽量にする必要があります。
- **クリックで再生する音声あり動画**
  - 推奨ファイルサイズ：**約5MBから20MB**
  - 注意：これらの動画はユーザー操作を介して読み込まれるため、やや大きめのサイズも許容されます。動画の長さに応じて、2分以内の動画であれば20MB以下を目指してください。

## 圧縮ツール

[HandBrake](https://handbrake.fr/)：動画ファイルの変換と圧縮を行うための無料のオープンソースツールです。

### インストール

1. [公式サイト](https://handbrake.fr/downloads.php)からHandBrakeをダウンロードします。
2. お使いのオペレーティングシステムに応じたインストール手順に従ってください。

または

Homebrewでのインストール（macOS）：

```bash
brew install --cask handbrake
```

### 推奨設定

1. HandBrakeを開き、動画ファイルを読み込みます。
2. 「Preset」→「Web」→ **「Social 25MB 2 Minutes 540p60」** を選択します。
   - プリセットで希望よりも大きなファイルサイズになる場合は、より小さいサイズにするため **「Social 25MB 5 Minutes 360p60」** に変更できます。
   - ただし、解像度を下げると動画の品質に影響する可能性があることに注意してください。
3. 「Video」タブで以下を設定します：
   - Video Codec：H.264 (x264)
   - Framerate (FPS)：Same as source
4. 「Audio」タブで、オーディオコーデックがAAC、ビットレートが128 kbpsに設定されていることを確認します。
   - 動画に音声が不要な場合は、オーディオトラックを完全に削除して容量を節約できます。
5. 「Start Encode」をクリックして圧縮プロセスを開始します

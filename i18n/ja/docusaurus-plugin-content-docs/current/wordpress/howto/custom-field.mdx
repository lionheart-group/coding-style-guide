# カスタムフィールド

WordPressは、1つのWYSIWYGフィールドのみを用意していますが、多くのWordPressプロジェクトでは、WordPressを使用してウェブサイトを更新しやすくするために他の入力フィールドを準備する必要があります。そのため、カスタムフィールドを追加して、追加の入力フィールドを作成します。

ほとんどの場合、カスタムフィールドの管理にはプラグインを使用していますので、以下のプラグインページを確認してください。

- [Smart Custom Fields](/docs/wordpress/plugins/custom-field/smart-custom-fields)
- [Advanced Custom Fields](/docs/wordpress/plugins/custom-field/advanced-custom-fields)

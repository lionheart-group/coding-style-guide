# お問い合わせフォーム

WordPressでお問い合わせフォームを実装する必要がある場合、主に以下の2つのプラグインのいずれかを使用しています。詳細情報はそれぞれのプラグインページを確認してください。

- [MW WP Form](/docs/wordpress/plugins/contact-form/mw-wp-form) : 基本的には、このお問い合わせフォームプラグインを使用してください
- [Contact Form 7](/docs/wordpress/plugins/contact-form/contact-form-7)

## 開発中

- [Template Oriented Form Utilities](https://github.com/lionheart-group/template-oriented-form-utilities) : これは、WordPressテンプレートでお問い合わせフォームを作成するためのライブラリです。まだ実装されていませんが、将来的に実装される予定です。

# 基本投稿タイプの利用

WordPressの基本投稿タイプを使用する場合は、`archive.php`と`single.php`のテンプレートファイルを更新してください。

## アーカイブページのURLを変更する方法

LH WordPressテンプレートは、基本投稿タイプのアーカイブURLを変更することができます。基本投稿タイプのアーカイブURLを変更する必要がある場合は、`config.php`を確認してください。

次の関数の値を変更した後、

```php
<?php

/**
 * 投稿アーカイブのスラッグ
 * Slug of post archive URL
 */
function get_post_slug() {
    return 'news';
}
```

`設定` -> `パーマリンク`ページを表示する必要があります。（このページを表示すると、WordPressはパーマリンク設定を更新します）

## 詳細ページのURLを変更する方法

基本投稿タイプの詳細ページのURLは、WordPressの管理画面の`設定` -> `パーマリンク`ページから変更できます。

## インストール

下記URLより**JetBrains ToolBox App**をインストールし、そちらから**PHPStorm**をインストールすることが可能です。

:::info M1 macを利用している場合
「.dmg (macOS Apple Silicon)」を選択してからダウンロードしてください
:::

https://www.jetbrains.com/toolbox-app/


## キーマップの設定

これまでVSCodeを利用していた場合、VSCodeになるべく寄せるためにKeymapの設定を変更するのをオススメしています。

1. 設定画面を開く (`⌘ + ,`)
2. 左メニューの「Kepmap」を選択
3. セレクトボックスの中の「VSCode (macOS)」を選択
4. (任意) 変更が必要なキーマップがあれば変更します
5. 画面右下の「OK」ボタンをクリック
